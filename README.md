# Code Flexers 팀 소개 웹사이트
이 프로젝트는 팀원들의 포트폴리오를 소개하고 팀을 소개하는 웹사이트를 만들어 취업 준비 또는 이직에 도움이 되기 위한 프로젝트입니다.  
## 편집 방법
아래 파일을 편집하여 본인의 포트폴리오를 작성하면 됩니다.
`/app/portfolio/본인이름/page.tsx`  

이미지는 `public/본인이름/..`안에 넣고 사용하시면 됩니다.  

새로운 페이지를 만들고 싶다면 `/app/portfolio/본인이름/..`내부 폴더로 만드실 수 있습니다.  

각자 이름의 브랜치를 만들고 작업해주세요.
